1. Proyecto IA Aplicada a Salud



Repositorio profesional de Marcos Muñoz.  



Contiene experimentos y proyectos relacionados con IA aplicada a imagen médica, modelos generativos, explicabilidad y VR/AR.



2\. Objetivos

* Clasificación y análisis de imágenes médicas (rayos X, TAC, resonancia).  
* Modelos generativos para asistencia médica e informes clínicos.  
* Visualización y simulación en VR/AR para salud y rehabilitación.
